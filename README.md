### 🔥 炼丹系统

| 指令                      | 说明                                                             |
| ------------------------- | ---------------------------------------------------------------- |
| `丹药配方`                | 查看所有可用配方，显示炼丹职业信息                               |
| `炼丹 <丹药名称>`         | 炼制丹药（成功后存入丹药背包）                                   |
| `查询配方 <丹药名称或ID>` | 查询配方详情（材料、等级要求、成功率、成本）                     |

| `炼丹信息`                | 查看炼丹职业信息（等级、称号、经验、成功率加成）                 |

**炼丹职业系统**

- 独立的炼丹职业系统，不依赖修炼境界
- 炼丹等级：0-100级，每10级获得新称号
- 成功率加成：每级+0.5%（最高+50%）
- 经验获取：炼丹成功获得经验，失败获得1/3经验
- 11个炼丹称号：见习炼丹师 → 初级炼丹师 → 中级炼丹师 → 高级炼丹师 → 炼丹大师 → 炼丹宗师 → 炼丹圣手 → 丹道真人 → 丹圣 → 丹帝 → 丹神

**配方系统**

- 59个完整配方（33种常规丹药 + 26种突破丹药）
- 配方包含：所需材料、炼丹等级要求、基础成功率、材料成本
- 材料成本为丹药售价的110%-130%，确保经济平衡
- 材料品质与丹药品质相匹配

| 指令                       | 说明                                               |
| -------------------------- | -------------------------------------------------- |
| `开垦灵田`                 | 首次创建3块田地（免费），之后每次增加2块（需灵石） |
| `我的灵田` / `灵田信息`    | 查看灵田状态和种植进度                             |

**灵田系统特性：**

- 初始免费获得3块田地，最多可扩展至15块
- 每次开垦增加2块田地，费用 = 当前田地数 × 10,000灵石
- 112种可种植药草，涵盖凡品到神品7个品级
- 不同品级药草成熟时间不同：凡品1-2小时，神品20天
- 种子解锁机制：购买同一种子5次后永久解锁，后续免费获取
- 收获数量随机2-4个，材料可用于炼丹

## 🎁 道具与丹药手册

| 类别        | 示例                               | 作用                                                 | 获取方式                                     |
| ----------- | ---------------------------------- | ---------------------------------------------------- | -------------------------------------------- |
| 破境丹      | 筑基丹、凝婴丹、化神丹             | 直接提升指定境界的突破成功率                         | 丹阁刷新、秘境稀有掉落                       |
| 修为丹      | 炼气丹、玄灵丹、九转灵丹           | 立即增加修为值（200-10万+）                          | 丹阁、炼丹系统、历练/秘境结算奖励            |
| 功能丹/灵丹 | 修炼加速丹、力量丹、灵智丹、回生丹 | 临时修炼倍率、永久属性增减、复活、定魂护盾等特殊效果 | 丹阁稀有刷新、Boss与高阶秘境掉落             |
| 材料        | 灵草、玄铁、星辰石、天材地宝       | 炼丹配方、悬赏任务、装备升级所需的基础资源           | 历练、秘境、悬赏、世界Boss掉落，灵田种植收获 |
| 法器/功法   | 青锋剑、玄铁甲、焚天诀             | 装备/心法提供物伤、法伤、防御与修炼倍率等长期加成    | 器阁 / 百宝阁刷新，高级Boss亦可掉落稀有法器  |

💡 使用 `物品信息 <名称>` 可随时查询任意上架物品或丹药的详细效果与推荐获取途径。

### 装备系统详解

**装备类型与属性**

- **武器（96种）**：提供物理伤害、法术伤害、防御、精神力等属性
- **法器/防具（10种）**：提供防御、气血上限、精神力等属性
- **功法（10种）**：提供全面属性加成和修炼倍率加成（exp_multiplier）

**装备属性显示**
- 玩家信息显示格式：`总属性(基础+装备)`
- 例如：`法伤：105(100+5)` 表示基础法伤100，装备加成5
- 战斗系统自动计算装备加成

**装备等级需求**
- 所有装备都有 `required_level_index` 字段，对应36境界系统
- 境界不足无法装备高级装备

### 丹药系统详解

**丹药分类**

1. **破境丹（27种）** - 专门用于突破境界
   - 位于 `pills.json` 配置文件
   - 提供突破成功率加成（breakthrough_rate）
   - 价格范围：10k灵石 - 200亿灵石
   - 品质：灵品 → 珍品 → 圣品 → 帝品 → 道品 → 仙品 → 神品

2. **增益丹（33种）** - 提升属性和修为
   - 位于 `items.json` 配置文件
   - 效果多样：恢复气血、增加修为、提升属性、突破加成等
   - 部分丹药有副作用（如焚天逆命增益丹扣除寿命）

**丹药效果显示**
- 效果：显示数值加成（如"恢复气血+100"、"突破成功率+40%"）
- 介绍：显示描述文本（如"辅助修士突破至筑基期的灵丹"）
- 储物戒和商店都会分别显示效果和介绍

**特殊丹药**
- **焚天逆命增益丹**：提升18%突破率，但会损失800气血和10年寿命
- **寿命系统**：部分丹药可增加/减少寿命，寿命变化会在服用时显示

### 储物戒系统详解

**储物戒等级**

储物戒按容量和品质分为8个等级，需要达到对应境界才能升级：

| 储物戒名称 | 品质 | 容量  | 需求境界 | 价格      |
| ---------- | ---- | ----- | -------- | --------- |
| 基础储物戒 | 凡品 | 20格  | 无       | 0（初始） |
| 灵玉储物戒 | 凡品 | 30格  | 筑基初期 | 1,000     |
| 玄铁储物戒 | 珍品 | 50格  | 金丹初期 | 5,000     |
| 星辰储物戒 | 珍品 | 80格  | 元婴初期 | 15,000    |
| 乾坤储物戒 | 圣品 | 120格 | 炼虚初期 | 50,000    |
| 须弥储物戒 | 圣品 | 200格 | 合体初期 | 200,000   |
| 混元储物戒 | 帝品 | 500格 | 大乘初期 | 1,000,000 |
| 太虚储物戒 | 神品 | 999格 | 地仙初期 | 9,999,999 |

**升级机制**
- 使用 `升级储物戒` 命令自动升级到下一级
- 系统自动检测境界和灵石是否满足条件
- 只能升级，不能降级或卸下
- 升级后容量立即生效，已有物品保留

**物品分类**
- 丹药：所有包含"丹"字的物品（不含内丹）
- 材料：草、铁、石、沙、参、芝、果、花、根、子、髓、核、粉、砂、皮、血、息、齿轮、碎片、精华、残页、遗物、种子、内丹
- 法器：剑、刀、枪、弓、幡、甲、袍、铠、阵
- 功法：功、诀、经、法
- 储物戒：储物戒类物品
- 其他：不属于以上分类的物品

**容量计算**
- 每种物品占1格，无论数量多少
- 例如：100个灵草只占1格，1个灵草也占1格
- 容量满时无法获得新种类物品，但可以增加已有物品数量

### 功法残篇合成系统

功法可以通过收集残篇自动合成完整版本，不同品质的功法需要的残篇数量不同：

| 功法品质  | 合成规则 | 功法示例                             |
| --------- | -------- | ------------------------------------ |
| 凡品      | 5个残篇  | 长春功、御风诀                       |
| 珍品      | 10个残篇 | 不动明王经、北冥神功、九阳神功       |
| 圣品/帝品 | 15个残篇 | 焚天诀、道经、吞天魔功、他化自在大法 |

**合成机制：**
- 当你获得功法残篇时，系统会自动检测是否集齐所需数量
- 集齐后自动消耗残篇并合成完整功法，无需手动操作
- 完整功法也可以直接掉落，但概率极低（鼓励通过收集残篇合成）
- 功法残篇主要通过秘境探索和历练系统的稀有掉落获得

## 🏗️ 技术架构

本插件采用 **Clean Architecture（整洁架构）** 设计：

| 层级                             | 职责                     | 示例                         |
| -------------------------------- | ------------------------ | ---------------------------- |
| **表现层（Presentation）**       | 处理用户命令和消息格式化 | PlayerHandler、CombatHandler |
| **应用层（Application）**        | 实现业务逻辑和用例       | PlayerService、CombatService |
| **领域层（Domain）**             | 定义核心业务模型和规则   | Player、Sect、Boss模型       |
| **基础设施层（Infrastructure）** | 提供数据持久化和外部服务 | PlayerRepository、Database   |

**架构优势：**

- 清晰的分层结构，易于维护和扩展
- 依赖注入容器管理对象生命周期
- 领域模型与数据库解耦
- 业务逻辑与表现逻辑分离
